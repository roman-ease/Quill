# Quill

Windows 向け Markdown エディタ。分割ビューでリアルタイムプレビューを確認しながら執筆できる、Electron 製のデスクトップアプリケーションです。

---

## デザイン思想

深夜の書斎をイメージしています。

窓の外は静かで、手元にはあたたかいコーヒー。古い木の机の上に開いたノートと、羽根ペン。余計なものは視界に入らず、ただ言葉と向き合える時間――Quill はそういう空間をモニターの中に再現したくて作られました。

デフォルトテーマの Default はその世界観の中心で、昼間の書斎に差し込む自然光のような清潔感のある色調を選んでいます。Light は白紙のノートのような完全な白基調で、PDF エクスポートと同じ見た目を画面上でも再現します。Sepia は紙とインクの質感に近い温かみ、Dark は照明を落とした夜の書斎。Terminal は機械の静寂、Vaporwave は夜の別の顔として、気分や作業の種類に合わせて部屋の雰囲気を変えるように使い分けることを想定しています。

UI が主張しすぎないのも意図的です。ツールバーはフォーカスモードで隠れ、必要なときだけ浮かび上がる。画面の大半を占めるのは、書いているテキストとそのプレビューだけ。アプリ名の Quill（羽根ペン）は、どれだけ機能が増えても「書く道具」であり続けることへの戒めでもあります。

## 特徴

### エディタ
- **リアルタイムプレビュー** — 分割ビュー / プレビューのみ を切り替え可能
- **複数タブ** — タブを自由に開閉、前回のセッションを起動時に自動復元
- **シンタックスハイライト** — highlight.js による 190 言語以上のコードブロック色付け
- **検索・置換** — 正規表現対応、マッチ件数リアルタイム表示
- **同期スクロール** — エディタとプレビューのスクロール位置を連動
- **TSV 貼り付け変換** — Excel / TSV をそのまま貼り付けると Markdown テーブルに自動変換
- **スペルチェック** — OS のスペルチェック機能をエディタに統合（設定で切替）

### プレビュー
- **Mermaid ダイアグラム** — フローチャート・シーケンス図・クラス図・ER 図・状態遷移図・ガントチャート・パイチャート・マインドマップ・タイムライン・XY チャート・Git グラフ・ユーザージャーニーをコードから描画
- **KaTeX 数式** — `$...$`（インライン）/ `$$...$$`（ブロック）記法で LaTeX 数式をレンダリング（設定で有効化）
- **画像ライトボックス** — プレビュー内の画像をクリックで拡大表示
- **コードコピーボタン** — コードブロックにワンクリックコピーボタンを表示
- **外部リンク制御** — 外部リンクをデフォルトブラウザで開く（設定で切替）

### 書式ツール
- **テーブル挿入** — 行・列数を指定してダイアログから挿入
- **目次生成** — 見出しから TOC を自動生成してカーソル位置に挿入
- **画像挿入** — ダイアログから URL / パスを指定して挿入
- **Mermaid テンプレート** — ツールバーアイコンまたはメニューから 12 種類のダイアグラムテンプレートを挿入
- **ユーザー定義テンプレート** — 現在のタブを名前を付けてテンプレートとして保存し、ツールバーから呼び出してカーソル位置に追記

### 外観・操作
- **アウトラインドロワー** — 見出し一覧をオーバーレイパネルで表示。クリックでプレビューをスムーズスクロール。スクロールに追従してアクティブ見出しをハイライト。`Ctrl+Shift+O` またはツールバーボタンでトグル
- **6 種類のテーマ** — Default / Light / Dark / Sepia / Vaporwave / Terminal
- **起動時テーマ** — 「前回のテーマを引き継ぐ」または特定テーマに固定する設定
- **フォーカスモード** — ツールバーを隠して執筆に集中（ホバーで一時表示）
- **ズーム** — `Ctrl++` / `Ctrl+-` / `Ctrl+0` でウィンドウ全体の表示倍率を調整
- **常に手前に表示** — 設定でウィンドウを最前面に固定
- **ウィンドウ状態記憶** — サイズ・位置・最大化状態を次回起動時に復元

### ファイル管理
- **エクスポート** — HTML・PDF への出力
- **ファイル監視** — 外部で編集されたファイルを自動検知して再読み込み通知
- **自動保存** — 設定した間隔で自動的に保存
- **最近使ったファイル** — ツールバーの履歴メニューからすぐにアクセス
- **エンコーディング / 行末コード** — UTF-8 / Shift_JIS 等、LF / CRLF を設定可能

### カスタマイズ
- **ショートカットキーカスタマイズ** — 主要な操作のショートカットを設定画面から変更・リセット
- **フォント / フォントサイズ / 行間** — エディタ・プレビューそれぞれ個別設定
- **シンタックスハイライトテーマ** — 5 種類のカラースキームから選択
- **Mermaid テーマ** — アプリテーマに連動する auto または個別指定

## インストール

### Windows

[Releases](../../releases) ページから最新の `Quill Setup X.X.X.exe` をダウンロードして実行してください。

- インストール先ディレクトリを任意に選択できます
- `.md` / `.markdown` ファイルの関連付けが自動で設定されます
- **動作要件:** Windows 10 以上（x64）

### macOS

[Releases](../../releases) ページから最新の `Quill-X.X.X.dmg` をダウンロードし、ディスクイメージを開いて Quill.app をアプリケーションフォルダへドラッグしてください。

- Intel Mac（x64）および Apple Silicon Mac（arm64）に対応しています
- 初回起動時に「開発元を確認できない」と表示された場合は、Finder でアプリを右クリック → 「開く」を選択してください
- **動作要件:** macOS 11 Big Sur 以上

## キーボードショートカット

デフォルトのショートカット一覧です。設定画面の「ショートカット」タブから変更できます（一部を除く）。

### ファイル

| 操作 | ショートカット |
|------|--------------|
| 新規ファイル | `Ctrl+N` |
| ファイルを開く | `Ctrl+O` |
| 上書き保存 | `Ctrl+S` |
| 名前を付けて保存 | `Ctrl+Shift+S` |
| 再読み込み | `Ctrl+R` |
| タブを閉じる | `Ctrl+W` |

### タブ

| 操作 | ショートカット |
|------|--------------|
| 新規タブ | `Ctrl+T` |
| 次のタブ | `Ctrl+Tab` |
| 前のタブ | `Ctrl+Shift+Tab` |

### 編集

| 操作 | ショートカット |
|------|--------------|
| 検索 | `Ctrl+F` |
| 検索と置換 | `Ctrl+H` |
| 太字 | `Ctrl+B` |
| 斜体 | `Ctrl+I` |
| リンク挿入 | `Ctrl+K` |
| テーブル挿入 | `Ctrl+Shift+T` |
| 目次を生成 | `Ctrl+Shift+C` |

### 表示

| 操作 | ショートカット |
|------|--------------|
| アウトライン | `Ctrl+Shift+O` |
| フォーカスモード | `Ctrl+Shift+F` |
| ズームイン | `Ctrl++` |
| ズームアウト | `Ctrl+-` |
| ズームリセット | `Ctrl+0` |

### ヘルプ

| 操作 | ショートカット |
|------|--------------|
| ショートカット一覧 | `F1` |

## 設定項目

設定画面（ツールバーの歯車アイコン）から以下を変更できます。

| カテゴリ | 設定項目 |
|---------|---------|
| **外観** | テーマ、起動時デフォルトテーマ、シンタックスハイライトテーマ、Mermaid テーマ |
| **エディタ** | フォント、フォントサイズ、行間、ワードラップ、タブサイズ、スペルチェック |
| **プレビュー** | フォントサイズ、同期スクロール、KaTeX 有効化、外部リンクをブラウザで開く |
| **ファイル** | 自動保存、自動保存間隔、画像保存モード、エンコーディング、行末コード、セッション復元 |
| **ウィンドウ** | ウィンドウサイズ記憶、常に手前に表示、表示倍率 |
| **ショートカット** | 各操作のショートカットキーを個別に変更・リセット、全リセット |

## 開発環境での実行

```bash
# 依存関係のインストール
npm install

# 通常起動
npm start

# 開発モード (DevTools 自動起動)
npm run dev
```

**動作要件:** Node.js 18 以上

---

## ビルド

```bash
npm run build
```

`dist/Quill Setup 1.1.3.exe` にインストーラーが生成されます。

## バージョン履歴

### v1.1.3 — 2026-04-23
**新機能**
- **ユーザー定義テンプレート** — 現在のタブの内容を名前を付けてテンプレートとして保存
  - ツールバーのテンプレートボタン（ブックマークアイコン）からドロップダウンで一覧を表示
  - テンプレート名をクリックするとカーソル位置に追記
  - ✕ ボタンで個別削除
  - テンプレートデータは `%APPDATA%\Quill\templates.json` に永続保存

**改善**
- **ツールバーのグループ整理** — ファイル操作 / 書式 / 挿入・テンプレート / 表示 / エディター動作 / 設定 の 6 グループに整理
  - グループ境界に太めのセパレーターを使用して視覚的に区別
  - 「最近使ったファイル」を右端から左側ファイル操作グループに移動
- **新規ファイルの保存時デフォルトファイル名** — 1 行目のテキストをファイル名として自動提案
  - 見出し記号（`#`）を除去し、Windows 禁止文字は `-` に置き換え
  - 1 行目が空の場合は従来通り「新規ファイル」

### v1.1.2 — 2026-04-22
**新機能**
- **タブ右クリックメニュー** — タブを右クリックでコンテキストメニューを表示
  - このタブを閉じる
  - 右のタブをすべて閉じる / 左のタブをすべて閉じる
  - 他のタブをすべて閉じる
  - すべてのタブを閉じる
  - 未保存ファイルがある場合は一括で保存確認ダイアログを表示

### v1.1.1 — 2026-04-22
**セキュリティ改善**
- `nodeIntegration: false` + `contextIsolation: true` に移行し、レンダラープロセスから Node.js への直接アクセスを遮断
  - `src/preload.js` を新設し、`contextBridge` 経由でホワイトリスト管理された IPC・API のみ公開
  - marked / DOMPurify / KaTeX をブラウザ UMD ビルドに切り替え
  - highlight.js は preload 経由で安全に公開

**バグ修正**
- クリップボードから画像を貼り付けた際、MIME タイプの解析が不正確で拡張子なしファイルが生成される問題を修正
- 未保存ファイルに貼り付けた絶対パス画像がプレビューに表示されない問題を修正

### v1.1.0 — 2026-04-22
**新機能**
- **アウトラインドロワー** — 見出しの階層構造をオーバーレイパネルとして表示
  - `Ctrl+Shift+O` またはツールバーボタン (☰) でトグル
  - ESC キーまたはドロワー外クリックで閉じる
  - 見出しクリックでプレビューをスムーズスクロール
  - スクロール位置に追従してアクティブ見出しをリアルタイムハイライト
  - ドキュメントの最上位見出しレベルを基準に相対インデント表示
- **テーマ追加: Default / Light 分離**
  - Default — 従来の Light テーマ（温かみのある羊皮紙調）
  - Light — 完全な白基調テーマ。PDF エクスポートと同じ配色
- **Mermaid テンプレート拡充**
  - ツールバーアイコンからドロップダウンでテンプレートを選択・挿入できるように
  - テンプレートを 6 種類 → 12 種類に拡張（パイチャート・マインドマップ・タイムライン・XY チャート・Git グラフ・ユーザージャーニーを追加）
  - ネイティブメニュー「挿入 > Mermaid テンプレート」もサブメニュー形式で全種類に対応

**バグ修正**
- Mermaid ダイアグラムがコードとして表示される問題を修正
  - v11 の SVG に含まれる CSS クラス定義 (`.error-icon`) を誤ってエラーと判定していた
- KaTeX 数式のスタイルが適用されない問題を修正
  - Electron レンダラーでの `__dirname` の挙動の違いにより CSS ファイルパスが解決できなかった

### v1.0.1 — 2026-04-20
**バグ修正**
- 目次自動生成で日本語のみの見出しのアンカーが空になり、リンクが機能しない問題を修正
  - ID 生成の正規表現を `[^\w\s-]` から `[^\p{L}\p{N}\s-]` (Unicode 対応) に変更
  - プレビューの見出し ID 生成と目次ジェネレーターで同じロジックを使用するよう統一
