# Quill

Windows 向け Markdown エディタ。分割ビューでリアルタイムプレビューを確認しながら執筆できる、Electron 製のデスクトップアプリケーションです。

---

## 特徴

### エディタ
- **リアルタイムプレビュー** — 分割ビュー / プレビューのみ を切り替え可能
- **複数タブ** — タブを自由に開閉、前回のセッションを起動時に自動復元
- **シンタックスハイライト** — highlight.js による 190 言語以上のコードブロック色付け
- **検索・置換** — 正規表現対応、マッチ件数リアルタイム表示
- **同期スクロール** — エディタとプレビューのスクロール位置を連動
- **TSV 貼り付け変換** — Excel / TSV をそのまま貼り付けると Markdown テーブルに自動変換
- **スペルチェック** — OS のスペルチェック機能をエディタに統合（設定で切替）

### プレビュー
- **Mermaid ダイアグラム** — フローチャート・シーケンス図・クラス図・ER 図・状態遷移図・ガントチャートをコードから描画、テンプレート挿入付き
- **KaTeX 数式** — `$...$`（インライン）/ `$$...$$`（ブロック）記法で LaTeX 数式をレンダリング（設定で有効化）
- **画像ライトボックス** — プレビュー内の画像をクリックで拡大表示
- **コードコピーボタン** — コードブロックにワンクリックコピーボタンを表示
- **外部リンク制御** — 外部リンクをデフォルトブラウザで開く（設定で切替）

### 書式ツール
- **テーブル挿入** — 行・列数を指定してダイアログから挿入
- **目次生成** — 見出しから TOC を自動生成してカーソル位置に挿入
- **画像挿入** — ダイアログから URL / パスを指定して挿入
- **Mermaid テンプレート** — ツールバーから 6 種類のダイアグラムテンプレートを挿入

### 外観・操作
- **5 種類のテーマ** — Dark / Light / Sepia / Vaporwave / Terminal
- **起動時テーマ** — 「前回のテーマを引き継ぐ」または特定テーマに固定する設定
- **フォーカスモード** — ツールバーを隠して執筆に集中（ホバーで一時表示）
- **ズーム** — `Ctrl++` / `Ctrl+-` / `Ctrl+0` でウィンドウ全体の表示倍率を調整
- **常に手前に表示** — 設定でウィンドウを最前面に固定
- **ウィンドウ状態記憶** — サイズ・位置・最大化状態を次回起動時に復元

### ファイル管理
- **エクスポート** — HTML・PDF への出力
- **ファイル監視** — 外部で編集されたファイルを自動検知して再読み込み通知
- **自動保存** — 設定した間隔で自動的に保存
- **最近使ったファイル** — ツールバーの履歴メニューからすぐにアクセス
- **エンコーディング / 行末コード** — UTF-8 / Shift_JIS 等、LF / CRLF を設定可能

### カスタマイズ
- **ショートカットキーカスタマイズ** — 主要な操作のショートカットを設定画面から変更・リセット
- **フォント / フォントサイズ / 行間** — エディタ・プレビューそれぞれ個別設定
- **シンタックスハイライトテーマ** — 5 種類のカラースキームから選択
- **Mermaid テーマ** — アプリテーマに連動する auto または個別指定

## インストール

[Releases](../../releases) ページから最新の `Quill Setup 1.0.0.exe` をダウンロードして実行してください。

- インストール先ディレクトリを任意に選択できます
- `.md` / `.markdown` ファイルの関連付けが自動で設定されます
- **動作要件:** Windows 10 以上（x64）

---

## キーボードショートカット

デフォルトのショートカット一覧です。設定画面の「ショートカット」タブから変更できます（一部を除く）。

### ファイル

| 操作 | ショートカット |
|------|--------------|
| 新規ファイル | `Ctrl+N` |
| ファイルを開く | `Ctrl+O` |
| 上書き保存 | `Ctrl+S` |
| 名前を付けて保存 | `Ctrl+Shift+S` |
| 再読み込み | `Ctrl+R` |
| タブを閉じる | `Ctrl+W` |

### タブ

| 操作 | ショートカット |
|------|--------------|
| 新規タブ | `Ctrl+T` |
| 次のタブ | `Ctrl+Tab` |
| 前のタブ | `Ctrl+Shift+Tab` |

### 編集

| 操作 | ショートカット |
|------|--------------|
| 検索 | `Ctrl+F` |
| 検索と置換 | `Ctrl+H` |
| 太字 | `Ctrl+B` |
| 斜体 | `Ctrl+I` |
| リンク挿入 | `Ctrl+K` |
| テーブル挿入 | `Ctrl+Shift+T` |
| 目次を生成 | `Ctrl+Shift+C` |

### 表示

| 操作 | ショートカット |
|------|--------------|
| フォーカスモード | `Ctrl+Shift+F` |
| ズームイン | `Ctrl++` |
| ズームアウト | `Ctrl+-` |
| ズームリセット | `Ctrl+0` |

### ヘルプ

| 操作 | ショートカット |
|------|--------------|
| ショートカット一覧 | `F1` |

## 設定項目

設定画面（ツールバーの歯車アイコン）から以下を変更できます。

| カテゴリ | 設定項目 |
|---------|---------|
| **外観** | テーマ、起動時デフォルトテーマ、シンタックスハイライトテーマ、Mermaid テーマ |
| **エディタ** | フォント、フォントサイズ、行間、ワードラップ、タブサイズ、スペルチェック |
| **プレビュー** | フォントサイズ、同期スクロール、KaTeX 有効化、外部リンクをブラウザで開く |
| **ファイル** | 自動保存、自動保存間隔、画像保存モード、エンコーディング、行末コード、セッション復元 |
| **ウィンドウ** | ウィンドウサイズ記憶、常に手前に表示、表示倍率 |
| **ショートカット** | 各操作のショートカットキーを個別に変更・リセット、全リセット |

## 開発環境での実行

```bash
# 依存関係のインストール
npm install

# 通常起動
npm start

# 開発モード (DevTools 自動起動)
npm run dev
```

**動作要件:** Node.js 18 以上

---

## ビルド

```bash
npm run build
```

`dist/Quill Setup 1.0.0.exe` にインストーラーが生成されます。
